# CSQL

A simple relational database management system (RDBMS).

This project is for educational purposes only.

## Building

Testing has only been done on linux machines.

Run the command below to build everything.

```./build.sh```

## Supports

[x] Declaring tables with a few column data types
[x] CRUD operations
[x] Joining

[] Basic indexing
[] Primary and unique keying

[x] SQL Interface
[x] Interactive Repl Mode

## Also included

[] Web App demo
