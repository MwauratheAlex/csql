# CSQL

A simple relational database management system (RDBMS).

## Supports

[] Declaring tables with a few column data types
[] CRUD operations
[] Basic indexing
[] Primary and unique keying
[] Joining

[] SQL Interface
[] Interactive Repl Mode

## Also included

[] Web App demo
